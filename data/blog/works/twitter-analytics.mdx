This is a web application that I built with my team for [COMP90024 - Cluster and Cloud Computing](https://handbook.unimelb.edu.au/subjects/comp90024) subject at my master course.
This system is an end-to-end system from harvesting the tweets, processing the data and display the result in the web.

## Technology Stack

- Python with Twitter API
- CouchDB
- Ansible
- NodeJS
- React/Redux
